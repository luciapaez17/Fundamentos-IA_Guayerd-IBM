## Descripción del proyecto: Tienda Aurelion

El proyecto consiste en el análisis integral de los datos comerciales de **Tienda Aurelion**, una tienda ficticia utilizada como caso práctico.  
El objetivo fue transformar datos crudos en información accionable mediante:

- Análisis exploratorio (EDA)  
- Modelos de Machine Learning  
- Visualización ejecutiva en Power BI  
- Un programa interactivo en Python para navegar resultados  

---

## Desarrollo del proyecto (sprints)

El trabajo se organizó en **cuatro sprints**, cada uno enfocado en una etapa distinta del proceso de análisis y modelado.

### Sprint 1 – Definición del problema y documentación inicial
- Planteo del problema y objetivos del proyecto.  
- Descripción del dataset y sus características.  
- Pseudocódigo y estructura general del programa interactivo.  
- Primera versión del programa en Python.

### Sprint 2 – Limpieza de datos y análisis exploratorio
- Preparación del dataset para análisis.  
- Estadística descriptiva y análisis de correlaciones.  
- Identificación de patrones, distribuciones y outliers.  
- Elaboración de gráficos representativos y actualización de la documentación.

### Sprint 3 – Modelado con Machine Learning
- Definición del objetivo del modelo (predicción y clasificación).  
- Entrenamiento y evaluación de modelos con scikit-learn.  
- Métricas, visualizaciones y análisis de resultados.  
- Integración de la información del modelo en el programa interactivo.

### Sprint 4 – Dashboard en Power BI
- Construcción del dashboard ejecutivo.  
- Creación de medidas, KPIs y análisis temporal.  
- Diseño del modelo relacional y narrativa visual.  
- Presentación del dashboard final y exposición de los resultados del análisis.

## Contenido del repositorio

El repositorio incluye los elementos principales del proyecto:

- **documentacion.md**: documento general con el análisis exploratorio, resultados, gráficos e interpretación.  
- **graficos/**: carpeta con gráficos generados durante el análisis y utilizados en la documentación.  
- **programa_interactivo.py**: aplicación en Python que permite navegar la documentación y visualizar gráficos desde la terminal.  
- **powerbi/**: carpeta con el dashboard exportado en PDF y el modelo relacional para una visualización rápida sin necesidad de Power BI.



---

## Programa interactivo en Python

Permite explorar:

- Documentación del proyecto  
- Gráficos del análisis exploratorio  
- Resultados de los modelos  
- Información sobre el dashboard en Power BI  


### Ejecución
```
python programa_interactivo.py
```
